# LoginSeguro
Projeto Spring Boot que usa Spring Security, BootStrap, Thymeleaf, Lombok e H2.

- [X] Spring Boot
- [X] Spring Security
- [X] Thymeleaf
- [X] BootStrap
- [X] Lombok
- [X] Controle de acesso com users in memory

Versão 2
- [ ] Acesso controle de acesso via JPA
